# Real-Time Toxicity Detection in Online Communication Platforms

This project focuses on detecting toxic or abusive messages in online chats using Natural Language Processing (NLP) and Machine Learning techniques.

Initially, the system was designed to filter toxic messages in online gaming chats, where abusive language is a common issue. Later, the scope was expanded to support general communication platforms such as social media, forums, and online discussion systems.

The main goal of this project is to classify messages as toxic or non-toxic in order to promote safer and healthier online communication environments.

---

## Objectives

* Identify harmful or abusive chat messages
* Classify messages based on toxicity
* Apply NLP techniques for text preprocessing
* Build a simple and interactive user interface for testing
* Support safer communication across online platforms
